Contexte

Explication du principe :

  • HTML : La structure HTML contient un titre (h1), un bouton et une div vide où le message de salutation sera affiché.
  • CSS : Le CSS stylise la page en donnant une mise en forme de base au contenu et en rendant le bouton attrayant.
  • JavaScript : Le JavaScript gère l'interaction utilisateur en ajoutant un écouteur d'événements au bouton. Lorsque le bouton est cliqué, il met à jour le contenu de la div avec le message de salutation.

Compétences acquises :

  • HTML : Comprendre et utiliser la structure de base d'une page HTML.
  • CSS : Appliquer des styles et mettre en forme les éléments HTML.
  • JavaScript : Ajouter de l'interactivité à une page web en répondant aux événements utilisateur.

Compétences transversales :

  • Compréhension des interactions entre HTML, CSS et JavaScript pour créer une expérience utilisateur cohérente et agréable.

Sous-compétences :

  • HTML : Utilisation des balises, gestion de l'arborescence des éléments.
  • CSS : Utilisation des sélecteurs, propriétés de style, mise en page.
  • JavaScript : Manipulation du DOM, gestion des événements, utilisation des fonctions.
